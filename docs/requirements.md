# 要求仕様書

## 1. 概要
このプロジェクトの目的は下記のとおりです。
- プロマネの技術を学ぶこと。
- 家電の操作性を上げること。

背景としてあるのは下記のとおりです。
- aws、teraform、gpt、docker、オブジェクト指向、python、電子回路、git、ci/cd、raspberrypiを学びたい
- 家電の操作を楽にしたい

## 2. 利用者
- 家電を便利に使用したいユーザ
